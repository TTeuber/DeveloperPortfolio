Selected Projects

GestureSynth — polyphonic synthesizer plugin github

Juno-inspired subtractive synth built from scratch in C++20/JUCE 8, shipping as Standalone, AU, VST3, AUv3, and CLAP. All DSP hand-written: PolyBLEP anti-aliased 4×-oversampled oscillators with a morphing sub-oscillator, resonant state-variable filter, Juno-style chorus, bucket-brigade-style delay, 16-line feedback-delay-network reverb. Novel drag-gesture modulation routing with a live-rendered modulation graph. Real-time-safe throughout: lock-free UI↔audio messaging, cached atomics, no locks or allocations in processBlock. Catch2 tests/benchmarks, pluginval validation, cross-platform CI.

PistompPedalboard — live multi-effect pedalboard on Raspberry Pi 5 github

C++17 pedalboard I'm building as my own live performance rig — engineered to real-world reliability requirements, not as a demo. One binary drives 30+ effects on juce::dsp, a Neural Amp Modeler block running .nam amp captures in realtime, the on-device LVGL screen/encoders/footswitches, and a Svelte web control surface kept live-consistent over Server-Sent Events. Chain edits are lock-free via epoch-based RCU, so effects can be added and reordered mid-performance without a click — verified by a TSan/ASan-swept stress test. Rigs/presets/setlists as JSON with atomic writes, hardware-free Catch2 DSP invariant suite, containerized arm64 build/deploy pipeline.

pistomp-hal — hardware abstraction layer + simulator github

JUCE-free C++17 HAL for the pi-Stomp v3 platform, extracted from the pedalboard and consumed as a static library via CMake FetchContent. Packages rotary encoders (libgpiod), SPI footswitches, NeoPixel LEDs, an ILI9341 TFT driven by LVGL v9, ADC metering, and a realtime duplex ALSA path (SCHED_FIFO, xrun recovery) behind clean headers — the Board class owns the shared SPI mutex so consumers can't corrupt the bus. The same headers are backed by a macOS simulator (LVGL in SDL2, CoreAudio/miniaudio), so an entire device app can be developed with no hardware attached.

SelfControl — Android content blocker github

Kotlin 2.0 / Jetpack Compose app using Device Owner APIs (the enterprise-MDM layer) to suspend apps at the OS level and push Chrome managed configurations. Add-only lock semantics enforced in a single LockManager; unit-tested domain logic and CI.
