Selected Projects

GestureSynth — polyphonic synthesizer plugin github

Gesture-driven, Juno-inspired subtractive synth written from scratch in C++20/JUCE 8, shipping as Standalone, AU, VST3, AUv3, and CLAP. All DSP hand-written: 4×-oversampled PolyBLEP DCO with morphing sub-oscillator, resonant state-variable filter, Juno-style chorus, bucket-brigade-style delay, 16-line FDN reverb. Defining feature: drag any modulation source (4 LFOs, 4 curve-shaped envelopes, MIDI expression) onto any knob, with live connector lines visualizing the whole modulation graph. Real-time-safe audio path, Catch2 tests/benchmarks, pluginval, 3-OS CI.

PistompPedalboard — live multi-effect guitar pedalboard, Raspberry Pi 5 github

C++17 pedalboard I'm building as my own live rig: 30+ effects (delays, reverbs, modulation, drives, dynamics, EQ, tuner) on juce::dsp, a Neural Amp Modeler block running .nam captures in realtime, on-device LVGL UI, and a phone-friendly Svelte control surface synced over Server-Sent Events. Lock-free chain editing via epoch-based RCU lets effects be reordered mid-performance without a click (TSan/ASan-verified stress test). Tap-tempo sync, JSON rigs/presets/setlists with atomic writes, hardware-free Catch2 DSP invariant suite.

StompLink — two-microcontroller guitar pedal github

Stereo effects pedal (overdrive, tremolo, delay) with firmware split across two MCUs: a Daisy Seed (STM32H750) runs the DSP at 48 kHz in interrupt context while an ESP32-S3 handles USB, WiFi, and a web UI served from the pedal itself. Knobs, OLED, and every connected browser stay in bidirectional sync over one protocol; the audio callback never takes a lock, and parameters are de-zippered per sample so edits are click-free. Tap tempo, host-tested protocol code, CI.

ArcadeSoundFX — browser-based 8-bit sound effects generator live demo · github

React 19 + TypeScript + Tone.js dual-oscillator synth (tone + noise, amp/pitch envelopes, vibrato LFO) that exports game-ready WAVs through a custom 16-bit PCM encoder — built so my coding students could add sound to their games. Includes an AI "describe a sound" feature (Claude API behind a rate-limited Cloudflare Worker) and shareable patch URLs.
