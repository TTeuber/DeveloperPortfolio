Selected Projects

HyprBlocker — tamper-resistant Linux content blocker github

Website and app blocker for Linux + Hyprland that treats "future me trying to cheat" as the adversary. A Python/FastAPI daemon (systemd) owns all enforcement and defends in depth: a self-healing mesh of 2–5 watchdog processes with obfuscated names that monitor the daemon and each other, NTP-verified time locks so changing the system clock can't end a block early, and a Manifest v3 browser extension whose heartbeat is tracked — any browser that stops reporting gets closed via Hyprland IPC. Wildcard/path-level URL rules with allow-list exceptions, per-weekday schedules, and read-only lock mode, managed from a React + TypeScript GUI in a native pywebview window. Fails closed on any error; pytest/ruff/bun CI.

Sermon Slides Generator — desktop document automation github

macOS desktop app that turns a list of text references into presentation-ready PDF slides in seconds, replacing a weekly manual copy-paste job — still used every week by the organization I built it for. Content is fetched with Requests + BeautifulSoup (polite rate limiting), cleaned, and split across slides at sentence boundaries; slides are rendered with Pillow (custom typography, optional QR-code title slide) and merged with pypdf. PyWebView UI plus a headless CLI, packaged by PyInstaller as a self-contained ~19 MB .app. pytest + ruff in CI.

Personal Lesson Agent — AI-powered learning platform github
